# 法人說明會 CSV 文件說明

## 📋 如何下載法說會 CSV 文件

### 步驟說明

1. **訪問公開資訊觀測站**
   - 網址：https://mopsov.twse.com.tw/mops/web/t100sb02_1
   - ⚠️ **注意**：網站載入可能需要一些時間，請耐心等待

2. **選擇查詢條件**
   - **市場別**：選擇「上市」（或「上櫃」）
   - **年度**：選擇民國年（例如：115年 = 2026年）
   - **月份**：選擇要查詢的月份（1月、2月、3月等）

3. **執行查詢**
   - 點擊「查詢」按鈕
   - 等待查詢結果顯示

4. **下載 CSV 文件**
   - 在查詢結果頁面，點擊「另存CSV」或「下載CSV」按鈕
   - 將文件保存到本項目的 `ir_csv` 文件夾中

5. **文件命名建議**
   - 建議將文件命名為：`1月.csv`、`2月.csv`、`3月.csv` 等
   - 或者保持原始文件名（系統會自動識別）

## 📁 文件放置位置

請將下載的 CSV 文件放置在以下目錄：

```
c:\trading_system\ir_csv\
```

### 文件命名建議

為了讓系統自動識別文件，建議使用以下命名方式：

- **方式1（推薦）**：按月份命名
  - `1月.csv` - 1月的法說會資料
  - `2月.csv` - 2月的法說會資料
  - `3月.csv` - 3月的法說會資料
  - 以此類推...

- **方式2**：保持原始文件名
  - 系統會自動嘗試從文件內容識別對應的月份
  - 但建議使用方式1以確保準確性

## 🔄 更新頻率建議

- 建議每月初更新當月及未來2個月的法說會資料
- 例如：1月初更新1月、2月、3月的資料

## ⚠️ 注意事項

1. CSV 文件必須使用 **Big5 編碼**（這是公開資訊觀測站的標準格式）
2. 確保文件完整下載，沒有損壞
3. 如果系統無法識別文件，請檢查文件名是否正確，或手動將文件重命名為 `1月.csv`、`2月.csv` 等

## 🆘 問題排除

如果法說會專區沒有顯示資料：

1. 檢查 `ir_csv` 文件夾是否存在且包含 CSV 文件
2. 確認 CSV 文件編碼為 Big5
3. 檢查文件是否包含正確的表頭（「公司代號」、「公司名稱」、「召開法說會日期」等）
4. 重啟 Flask 應用程式
