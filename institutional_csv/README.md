# 三大法人買賣超 CSV 說明

當系統無法從證交所自動取得資料時（例如連線、SSL 或網站阻擋），可改用手動下載的 CSV。

## 如何下載

1. 打開 [證交所 三大法人買賣金額統計表](https://www.twse.com.tw/zh/trading/foreign/bfi82u.html)
2. 選擇 **日報表**，選定 **日期**（例如 2026/01/02）
3. 點 **CSV 下載**，儲存檔案

## 放置位置與檔名

請將下載的 CSV 放到此資料夾：

```
c:\trading_system\institutional_csv\
```

檔名請用 **日期** 命名，例如：

- `20260102.csv` — 2026 年 1 月 2 日
- `20260103.csv` — 2026 年 1 月 3 日
- 或 `BFI82U_20260102.csv` 也可

放好後回到系統「三大法人買賣超專區」按 **更新** 即可顯示。

## 注意

- 需逐日下載（證交所一次只能選一個日期）
- 檔案編碼為 UTF-8 或 Big5 皆可
- 內容需為 BFI82U 報表格式（含「類別」「買賣超」等欄位）
