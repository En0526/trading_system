# 🚀 快速開始指南

## 三步驟啟動系統

### 步驟 1: 安裝依賴套件

```bash
pip install -r requirements.txt
```

或者使用環境設置腳本：

```bash
python setup.py
```

### 步驟 2: 啟動應用程式

```bash
python app.py
```

### 步驟 3: 打開瀏覽器

訪問: **http://localhost:5000**

---

## 📁 檔案說明（快速參考）

| 檔案 | 功能 |
|------|------|
| `app.py` | **主程式** - 啟動 Web 伺服器，定義所有 API |
| `config.py` | **配置檔** - 管理市場代碼、端口等設定 |
| `market_data/data_fetcher.py` | **數據獲取** - 從 yfinance 獲取市場數據 |
| `timing/timing_selector.py` | **擇時分析** - 判斷最佳交易時機 |
| `strategy/strategy_matcher.py` | **策略匹配** - 推薦最適合的交易策略 |
| `templates/index.html` | **前端頁面** - 市場監控介面 |
| `static/css/style.css` | **樣式表** - 網頁外觀 |
| `static/js/main.js` | **前端邏輯** - 數據載入和顯示 |

---

## 🎯 主要功能

1. **市場監控** - 自動顯示美股、台股、國際市場數據
2. **擇時分析** - 智能判斷市場時機（看漲/看跌/中性）
3. **策略推薦** - 根據市場環境推薦交易策略
4. **自動刷新** - 每 60 秒自動更新數據

---

## 🔧 常見問題

**Q: 如何新增監控的股票？**
A: 編輯 `config.py`，在對應的市場字典中新增標的代碼

**Q: 如何修改更新頻率？**
A: 修改 `static/js/main.js` 中的 `setInterval` 時間（單位：毫秒）

**Q: 如何自訂擇時邏輯？**
A: 編輯 `timing/timing_selector.py` 中的 `analyze_market_timing` 方法
